MAP

Permite recorrer un arreglo y crear uno nuevo transformando cada elemento con una función.


const numeros = [1, 2, 3];
const doble = numeros.map(n => n * 2);
console.log(doble); // [2, 4, 6]

FILTER

Filtra los elementos de un arreglo que cumplen una condición y devuelve un nuevo arreglo con ellos.


const edades = [12, 18, 22, 15];
const mayores = edades.filter(e => e >= 18);
console.log(mayores); // [18, 22]

REDUCE

Reduce todos los valores de un arreglo a un solo resultado (por ejemplo, la suma total).


const nums = [1, 2, 3, 4];
const suma = nums.reduce((a, b) => a + b, 0);
console.log(suma); // 10

FIND

Devuelve el primer elemento que cumple una condición.


const numeros = [10, 20, 30, 40];
const encontrado = numeros.find(n => n > 25);
console.log(encontrado); // 30

SORT

Ordena los elementos de un arreglo de forma alfabética o numérica.


const letras = ['b', 'a', 'c'];
letras.sort();
console.log(letras); // ['a', 'b', 'c']

INCLUDES

Verifica si un arreglo contiene un valor específico, devuelve true o false.


const frutas = ['manzana', 'pera', 'uva'];
console.log(frutas.includes('pera')); // true

INDEX OF

Devuelve la posición (índice) donde se encuentra un elemento dentro del arreglo.


const colores = ['rojo', 'verde', 'azul'];
console.log(colores.indexOf('verde')); // 1

SOME

Devuelve true si al menos uno de los elementos cumple la condición.


const edades = [15, 20, 17];
console.log(edades.some(e => e >= 18)); // true

EVERY

Devuelve true si todos los elementos cumplen la condición.


const edades = [18, 22, 25];
console.log(edades.every(e => e >= 18)); // true

SLICE

Copia una parte específica del arreglo sin modificar el original.


const animales = ['perro', 'gato', 'pez', 'ave'];
const copia = animales.slice(1, 3);
console.log(copia); // ['gato', 'pez']
